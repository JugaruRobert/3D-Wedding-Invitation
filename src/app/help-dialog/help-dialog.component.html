Aceasta invitatie se poate folosi intr-o maniera similara invitatiei fizice.

Plicul se poate deschide apasand pe partea din fata a acestuia si se poate inchide apasand pe partea de sus, situata deasupra biletului din interiorul acestuia.

Biletul se poate scoate din plic si adauga din nou in plic apasand oriunde pe suprafata lui.

(sub invitatie) - Roteste plicul sau biletul din interior, daca acesta este scos din plic.
(dreapta sus) - Ascunde invitatia pentru a se putea vedea mai bine pozele din fundal. De asemenea, face invitatia vizibila cand aceasta este ascunsa.
(dreapta sus) - Arata adresa locatiei evenimentului.
